Top 3 Risk Hotspots

1. billing ↔ orders — Connascence of Value

Monetary amounts flow without a shared value object. 12 cross-module value pairs. Fix: introduce Money value object.

2. OrderService — SRP Violation

Validates, prices, persists, and notifies. 4 reasons to change. Fix: split by responsibility: OrderValidator, OrderPricer, OrderRepository.

3. Stripe SDK leak — DIP Violation

3 modules depend directly on Stripe SDK. Fix: introduce PaymentGateway interface.
