A few months ago, I started building CogniCode because I was frustrated watching AI coding assistants read code like it was 1995.

Today, CogniCode has:

• **32 MCP tools** for code intelligence
• **6 languages** supported (Rust, Python, TypeScript, JavaScript, Go, Java)
• **763 tests** with sandbox orchestrator
• **Persistent graph cache** using embedded redb
• **Zero-config setup** for Claude Desktop, Cursor, Windsurf, OpenCode

The most rewarding part?

Watching an AI agent go from "I read 15 files and think this might work" to "analyze_impact shows medium risk in 3 modules, here's the exact call chain"

That transition — from reading code to querying it — is what CogniCode is all about.

Huge thanks to everyone who's tried it, filed issues, and contributed ideas.

GitHub: https://github.com/Rubentxu/CogniCode
